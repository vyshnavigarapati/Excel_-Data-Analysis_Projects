📊 Day 4 - E-commerce Dashboard (Excel)

🔍 Overview

Built an interactive Excel dashboard to analyze revenue, category performance, and return behavior.

🎯 Key KPIs

- Total Revenue: ₹64,300
- Total Orders: 10
- Average Order Value: ₹6,430
- Return Rate: 30%

📊 Features

- Pivot Tables for data analysis
- Interactive Slicers (Country, Category, Payment Method)
- Charts:
  - Revenue by Country
  - Revenue by Category
  - Monthly Revenue Trend
  - Payment Method Distribution

💡 Key Insights

- Electronics generates highest revenue with zero returns
- Beauty category has 100% return rate (critical issue)
- Overall return rate is high, impacting profitability

🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Data Visualization
